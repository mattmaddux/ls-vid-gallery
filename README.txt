A simple video gallery plugin for Wordpress.

== Description ==

This plugin allows users to add videos from YouTube or Vimeo to galleries, apply tags, and them display those galleries with shortcodes. It's meant to be simple and straightforward, so it does not offer many customization options.

== Installation ==

## Via Wordpress

1. In Wordpress Settings open "Plugins"
2. Click "Add New"
3. Search for "Lumberstack Video Gallery"
4. Click "Install"
5. Click "Activate"
6. Done!

## Manually

1. Visit Github Repository: https://github.com/mattmaddux/ls-vid-gallery
2. Click 'Code' -> 'Download ZIP'
3. In Wordpress Settings open "Plugins"
4. Click "Add New"
5. Click "Upload Plugin"
6. Select 'ls_vid_gallery.zip' and upload
7. CLick 'Activate'
8. Done!
